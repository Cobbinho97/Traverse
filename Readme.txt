Readme

This folder contains the full project of Andrew Cobb. It contains all the scripts and the Unity project
developed during the project.

Steps to follow in order to play the game:

1. Before being able to play the game, Unity Remote 5 must be downloaded to a mobile device,
with a usb connecting the mobile device to the computer.

2. In order to play the game, the KV6003 - Individual Computing project folder
must be unzipped onto a computer. 

3. Upon being unzipped the project should be opened in unity on the computer. 

4. Before pressing the play button in Unity, make sure Unity remote 5 is opened on the mobile device so that upon 
pressing the play button the game will load onto the mobile device.

5. The player must then navigate through each level until the final level is completed, then the game can
be closed.
